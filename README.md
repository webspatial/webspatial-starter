# WebSpatial Starter

This repository is no longer the recommended way to start a WebSpatial project.

Instead of cloning this repo, use [`@webspatial/starter`](https://www.npmjs.com/package/@webspatial/starter) to generate what you need directly.

## Recommended Workflow

### Create a New WebSpatial Project

```bash
npx @webspatial/starter create my-webspatial-app
```

### Add WebSpatial AI Resources to an Existing Web Project

Run this inside your current web project:

```bash
npx @webspatial/starter ai
```

The `ai` command generates project-local WebSpatial AI resources for the current project, including documentation, project guidance, and agent skills, so AI coding agents can help with setup, integration, and follow-up development.
